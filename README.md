# Assets

## Descrição

Este repositório centraliza `recursos visuais` utilizados em projetos da organização.

A ideia é manter um local único, organizado e confiável para reutilização de assets visuais em aplicações, sites, documentação e materiais institucionais.

## Objetivo

 - Centralizar logos e outros assets visuais
 - Garantir consistência visual entre projetos
 - Facilitar o acesso e a reutilização de recursos
